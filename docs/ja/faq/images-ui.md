# 画像表示

商品画像は「顧客が注文する気になるかどうか」の鍵を握りますが、同時に元のファイルサイズに足をすくわれやすいポイントでもあります。

## 商品画像が表示されない、または読み込みが非常に遅いですか？

以下の順序で対処してみてください：

1.  画像ファイル名に特殊記号が含まれていないか確認します（**純粋な数字または英字**で命名することをお勧めします）。
2.  ローカルネットワークの伝送帯域は限られているため、1枚の画像を **500KB以内** に圧縮することをお勧めします。
    10MBもある高解像度の元画像を直接アップロードしないでください（現場では非常に苦労します）。

## インターフェイスのスタイルをカスタマイズできますか？

現時点では利便性のため、ライト/ダークモードとテーマカラーのみカスタマイズ可能です。
より詳細なテーマカスタマイズ機能は、将来リリースされる予定です。
