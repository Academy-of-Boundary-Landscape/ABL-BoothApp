<summary><h2 id="ai-拍照识别主要更新功能">📸 AI 拍照识别（主要更新功能）</h2></summary>

**把制品放在平板摄像头前面，就自动识别加入购物车。**

用户不用在列表里翻找，摊主也不必贴大量条形码。

这个功能是为**同人摊的真实场景**设计的：
- **二次元制品种类多、单品量少**——贴和打印条形码太麻烦太累
- **帮朋友看摊 / 寄售**——面对几十种不熟悉的货物，拍照即可识别是什么
- **小件商品**（明信片、徽章、亚克力）——贴不上条码也不美观
- **宣传总要拍实物图**——既然拍了，不如用来进行识别

**开箱即用**：首次启用在控制台点击下载模型即可（建议展会前一天联网下好），兼容 Windows 和安卓设备，无需其他依赖。之后全程即可离线进行模型推理。

</details>

<summary><h2 id="界面和体验全面翻新">🎨 界面和体验全面翻新</h2></summary>

### 摊位平板（顾客看的页面）

这是顾客最直接感受我们软件的地方，我们把每个细节都进行了打磨：

- **闲置吸引屏**——平板没人动时自动显示欢迎画面，下一位顾客点屏幕进入
- **双模式入口**——顾客可以自己选择"浏览点单"或"拍照识别"
- **商品图比例从 1:1 改成 3:4**——更适合立绘、明信片、海报这些竖版制品，少留白
- **加购视觉反馈**——点击商品时卡片会回弹 + 高亮圈，确认"点到了"
- **售罄自动沉底**——还有货的商品排在前面，售罄的沉到底部，顾客少浪费时间
- **低库存预警**——剩货少时有琥珀色边框 + "仅剩 X 件"角标
- **结算二次确认**——防止误触大金额订单
- **支付倒计时**——加入付款码自动消失倒计时，前 60 秒显示细进度条，最后 30 秒出现倒计时；需要更多时间可以一键重置
- **购物车小缩略图**——购物车里的每件商品都有小图

### 管理端（摊主看到的页面）

- **统一的页面骨架**——控制台、展会管理、全局商品库风格一致
- **折叠面板**——内容多的页面可以折叠起来，聚焦当前在做的事
- **移动端大量修复**——iPhone 刘海 / 底部横条不再遮元素，手机端商品名能完整显示，购物车按钮按得更准

</details>

<details open>
<summary><h2 id="新人引导大幅完善">🧭 新人引导大幅完善</h2></summary>

v1.0 的一个反馈是"界面太复杂不知道点哪儿"，这版把**引导**做得尽可能全面：

### 首次启动欢迎页
刚装完打开软件，会有专门的 **"欢迎使用摊盒 / 进入管理后台开始配置"** 卡片。老用户自动跳过。

### 控制台"快速开始"进度条
告诉你"5 步到开业"——创建展会 → 添加商品 → 上架 → 启动展会 → 获取二维码。**哪步完成了自动打勾**，一目了然。

### 每页都有 `?` 帮助气泡
6 个管理页面标题旁都有 `?` 按钮，点一下弹出**这一页能干什么、怎么用**的要点说明。

### AI 拍照识别面板：行动式引导
- 加入如何操作的傻瓜式引导和跳转按钮
- 模型装好但没上传识别图 → 提示 **"去上传识别图"** 一键跳转
- 商品库新增 **"识别图"列**，🔴 未上传 / 🟡 1~2 张 / 🟢 3 张+ 一眼可见

</details>

<details open>
<summary><h2 id="标签tag系统">🏷️ 标签（tag）系统</h2></summary>

以前商品只有一个"分类"（明信片 / 亚克力 / 钥匙扣等），这版加入了**多维度的 tag 系统**：

```
  商品A：博丽灵梦 · 红色 · 东方Project · 新刊
  商品B：博丽灵梦 · 蓝色 · 东方Project
```

- 管理端：编辑商品时多选 / 输入标签，支持自定义新标签
- 顾客端：商品网格上方新增**标签快筛条**，点击按角色 / IP / 系列筛选，和侧边分类**取交集**
- 搜索框也能直接搜标签
- 数据包导入导出自动携带标签

</details>
