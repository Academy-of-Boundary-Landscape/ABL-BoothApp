# 网络连接

这类问题出现频率最高，但也最好解决：**99% 都是“没连同一个热点”或“被防火墙拦了”。**

:::tip 先按顺序排查（真的有效）
1. **同一个 Wi-Fi/热点**：主机端（电脑/平板）和手机必须连同一个热点  
2. **Windows 防火墙**：是否允许“摊盒/Booth-Kernel”通过？  
3. **热点重启导致 IP 变了**：重启热点后请在主机端 **重新生成二维码**
:::

## 我在APP里面找不到“扫码”按钮？

- 这是对的，我们的架构是:
    - **主机端（Host）**：运行 Tauri 桌面/移动应用，负责业务逻辑和数据存储的实际实现
    - **客户端（Client）**：局域网内任意设备通过浏览器访问主机端提供的网页界面
- 因此**扫码按钮在主机端应用里就是没有的**，请在主机端点击 **“生成二维码”** 按钮，
  然后用手机扫描二维码访问点单页面。

## 手机扫码后显示“连接超时”或网页无法打开？

这是最常见的问题，请按顺序排查：

1. **检查热点**：确保主机（电脑/平板）和扫码的手机连的是同一个 Wi-Fi/热点  
2. **防火墙拦截**：Windows 主机请检查防火墙是否允许“摊盒”通过，或尝试暂时关闭防火墙验证  
3. **IP 变动**：如果重启过热点，IP 地址可能会变，请在主机端点击 **“重新生成二维码”**

:::warning 小经验
漫展现场很多人会下意识连场馆 Wi-Fi，但那玩意儿经常抽风，还可能隔离设备互访。**热点局域网才是最稳的。**
:::

## 没有网络 / 信号差能用吗？

**完全可以。** 摊盒是“离线优先”的设计。  
推荐用一台设备开启 **手机热点** 组建局域网，其他设备连接它即可：

- **不需要连接互联网**
- **不消耗流量**

## 我是校园网 / 公共 Wi-Fi，设备连不上？

公共网络通常开启了 **AP 隔离**（禁止设备间互相访问）。  
解决办法也很简单：**请务必使用热点来组建网络**，这是漫展现场最稳妥的方案。

## 中途断网了，数据会丢吗？

**不会。** 只要主机端（App 端）没有关闭，数据就一直存在。  
重新连接网络后，手机端刷新页面即可恢复之前的状态。
