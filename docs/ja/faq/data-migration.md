# データの安全と移行

このページでは、以下の3つのことについて説明します：データはどこにあるのか、デバイスを変更する方法、Excelのエクスポートで何が得られるか。

## データはどこに保存されますか？ 安全ですか？

すべてのデータ（画像、会計記録、在庫情報）は、**親機デバイスのローカルデータベース**に保存されます。
当方にはクラウドサーバーは一切ありません。売上データは、あなた自身以外には誰にも知られることはありません。

## 新しいデバイスに変更する場合、どうやってデータを移行すればいいですか？

「グローバル商品マスタ」ページで、**「データパッケージをエクスポート」** をクリックすると、すべての商品情報と画像がパッケージ化されてダウンロードされます。
このファイルを新しいデバイスに送信し、設定ページで **「データをインポート」** を選択するだけです。

## エクスポートしたExcelファイルには何が含まれていますか？

イベント概要と、重要な**商品別販売数量統計表**が含まれています。
この表をそのままサークルの主催者に提出すればOKです（本当に手間が省けます）。
