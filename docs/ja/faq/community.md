# コミュニティとオープンソース

ここでは主に、ダウンロードチャネル、アップデート方法、フィードバック窓口、およびオープンソースに関するよくある質問を扱います。

## 最新版はどこでダウンロードできますか？

唯一の公式チャネルは **GitHub Releases ページ** です。
有料ダウンロードや「代行導入サービス」を要求するチャネルはすべて詐欺ですので、ご注意ください。

## 自動更新はできますか？

サイドバーの「更新を確認」ボタンをクリックすると、システムが最新バージョンを自動検出し、インストーラーをダウンロードするように案内します。
現在のバージョンではワンクリックでの自動更新はサポートされておらず、インストーラーを手動でダウンロードして上書きインストールする必要があります。

## バグに遭遇したり、提案がある場合はどこにフィードバックすればいいですか？

- プログラムエラー: **GitHubリポジトリ** で **Issue** を投稿することを強くお勧めします（開発者が修正を追跡しやすいため）。
- 使い方の疑問/雑談: **ユーザー交流グループ** への参加を歓迎します。

## 開発者ですが、プロジェクトにコードを貢献できますか？

**大歓迎です！**
小さなバグの修正でも、新しい機能モジュールの開発でも、あなたの貢献は多くの頒布者を助けることになります。
**Pull Request** の投稿をお待ちしています。

## ソフトウェアを改変して再配布（または販売）してもいいですか？

このプロジェクトは **MITライセンス** に基づいています。確かに、改変と再配布の自由はあります。
ただし、「同人頒布の参入障壁を下げる」ことを目的とした無料ツールとして、それを商業ソフトウェアとしてパッケージングして販売することは **強く推奨しません**。

原作者の情報を保持し、オープンソースの精神を尊重してください。
