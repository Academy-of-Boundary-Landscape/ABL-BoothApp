:::tip ✅ 离线优先：没有网络也能出摊
摊盒 **不依赖互联网**。用手机开热点（不耗流量）即可组网，多设备在局域网内实时同步。
:::

:::info 🔒 数据与隐私：只在你的设备里
所有数据（商品、订单、库存、图片）都保存在你的设备本地。**我们没有云端服务器，也不会收集营业额/交易数据**。
:::

:::warning 🧩 可持续使用：即使项目不更新，你也能一直用
数据在本地、功能离线可用；你导出的报表（Excel 等）也在自己手上。**不担心“服务停了就用不了”**。
:::

:::tip ⭐ 开源透明：可审计、可复用、可二次开发
项目 100% 开源（MIT 协议）。你可以查看代码、提交 Issue、贡献 PR，也可以 fork 后做适合自己社团的改造。
:::
