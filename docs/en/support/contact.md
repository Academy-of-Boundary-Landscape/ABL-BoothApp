# Contact & Support

Get help with BoothKernel.

## GitHub Issues

For bug reports, feature requests, or technical questions:

**[Open an Issue on GitHub](https://github.com/Academy-of-Boundary-Landscape/ABL-BoothApp/issues)**

When reporting bugs, please include:
- Device type and OS version
- BoothKernel version
- Steps to reproduce
- Screenshots if applicable

## Community

Join discussions on GitHub for:
- Usage tips and best practices

- General questions

## Contributing

BoothKernel is open source! Contributions welcome:

- **Code**: Submit PRs for bug fixes or features
- **Documentation**: Improve guides and translations
- **Testing**: Test beta releases and report issues

See [Contributing Guide](https://github.com/Academy-of-Boundary-Landscape/ABL-BoothApp/blob/main/CONTRIBUTING.md) for details.
