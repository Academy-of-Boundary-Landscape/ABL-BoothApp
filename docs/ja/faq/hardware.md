# ハードウェアの推奨事項

一言で言うと：Booth-Kernelは高い性能を要求しませんが、「継続的なオンライン状態」は重要です。そのため、親機端の選択が鍵となります。

## スマホで親機端を実行できますか？

技術的には可能ですが、**強くお勧めしません**。

現実的な理由があります：

-   画面ロックやバックグラウンドでプロセスが強制終了されやすいです。
-   発熱によりパフォーマンス低下や通信切断が起こりやすいです。
-   現場ではそれで決済したり連絡を取る必要もあり、誤操作が発生しやすいです。

**親機端にはノートPCまたはタブレットを使用することをお勧めします。**

## デバイスの性能に要件はありますか？

要求は非常に低いです：

-   最新版のChrome/Edgeが動作するWindows PCであれば何でも
-   またはAndroid 8.0+を搭載したタブレット/スマホ

これらはすべてスムーズに動作します。古いデバイスを「顧客注文端末」として再利用するのも良い方法です。

## 画面を常時点灯させておく必要がありますか？

-   親機端：画面を点灯させたままにするか、電源設定で**スリープ/休止状態にしない**ように設定することをお勧めします。
-   顧客端（タブレット）：**画面を常時点灯**させるように設定し、顧客が注文中に突然画面が暗くなるのを防ぐことをお勧めします。
