# 突発的なトラブル

慌てないでください。現場での頒布では、誤操作や注文ミス、デバイスのバッテリー切れなどはつきものです。ほとんどの場合「取り消し可能」です。

## 顧客が誤った注文をした / 注文をキャンセルしたい場合はどうすればいいですか？

店員端（スタッフ画面）で：

- **「完了」はタップしないでください**
- 直接、赤い **「キャンセル」** ボタンをタップしてください

その注文は破棄され、確保されていた在庫は自動的に戻ります。

## 「完了」を誤ってタップしてしまいました。取り消したい場合はどうすればいいですか？

慌てずに：

1.  親機端の「注文管理」ページに移動します。
2.  誤操作した注文を見つけます。
3.  ステータスを **「キャンセル済み」** に変更します。

システムは以下を行います：

-   在庫を戻します
-   売上統計を修正します

## 親機デバイスが突然バッテリー切れ / フリーズしてしまいました。どうすればいいですか？

Booth-KernelはSQLiteによる**リアルタイム永続化**ストレージを採用しています。
デバイスとソフトウェアを再起動すると、すべての商品情報、過去の注文、在庫データが、フリーズする直前の状態に **自動的に復元** されます。

:::tip 現場でのアドバイス
親機端にはモバイルバッテリーや電源タップを用意することをお勧めします。Booth-Kernelが不安定だからではなく、会場の電力供給自体が不安定な場合があるためです。
:::
